Simple JSON parser & builder

Author:Chihiro Ito
License:Public domain (unlike other files)
Support:groups.google.com/group/webos-goodies/

This file contains two simple JSON processing classes. JsonParser converts a JSON string to an array or a hash. JsonBuilder performs vice versa. These classes are standard compliant and are designed for stability and reliability. Especially JsonParser has UTF-8 validation functionality so you can avoid some kind of security attack.
